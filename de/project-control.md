1. Nicola arbeitet für einen grossen Konzern. Sie

   * glaubt nicht, dass es eine bessere Stelle gibt
   * mag ihre Arbeit, obwohl sie schwierig ist
   * wird irgendwann eine Gehaltserhöhung bekommen

2. Die Stelle bietet

   * ein hohes Gehalt
   * ein niedriges Gehalt
   * mehr Geld, wenn sie härter arbeitet

3. Nicolas Vorgesetzter

   * sagt, ihre Arbeit entspreche nicht seinen Anforderungen
   * möchte, dass sie neue Ideen entwickelt
   * bittet sie, mehr Diagramme in ihre Berichte einzubauen

4. Nicola erhält seltsame Textnachrichten, aus denen hervorgeht, dass

   * das Unternehmen seine Mitarbeitenden unter Drogen setzt
   * sie beobachtet wird
   * sie niemals befördert wird

5. Nicola findet eine Bombe, und die Nachrichten erklären ihr, wie sie sie benutzen soll:

   * die Knöpfe gedrückt halten, um einen Zehn-Minuten-Timer zu starten
   * die Drähte mit den roten Röhren verbinden
   * den Code ‚vier zwei sieben acht‘ eingeben

   Doch dann wacht sie auf. Sie hat nur geträumt.

6. Ihr Vorgesetzter entlässt sie, und Nicola erkennt

   * dass ihr die Stelle nicht guttat
   * dass sie niemals eine bessere Stelle finden wird
   * dass sie härter hätte arbeiten sollen
